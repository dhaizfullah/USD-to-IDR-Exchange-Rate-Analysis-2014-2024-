## Project Overview
This project presents a comprehensive 10-year longitudinal study analyzing the currency exchange dynamics between the US Dollar (USD) and the Indonesian Rupiah (IDR). Using a data-driven approach, this research identifies volatility trends and market patterns over a decade of financial fluctuations.

## Key Features & Metrics
The interactive dashboard developed in **Microsoft Power BI** provides deep insights into several critical financial indicators:
* **Dynamic Exchange Tracking:** Monitoring highest and lowest selling and buying rates from 2014 to 2024.
* **Spread Analysis:** Visualizing the difference (spread) between selling and buying rates to understand market liquidity.
* **Moving Averages:** Automated calculation of daily and yearly averages to smooth out volatility and identify long-term trends.
* **Historical Benchmarking:** Identifying peak appreciation and depreciation cycles of the Rupiah.

## Insights Derived
* **Volatility Trends:** Successfully detected significant patterns in exchange rate fluctuations, providing a clear overview of currency performance over time.
* **Data-Driven Conclusions:** Transformed raw historical data into actionable insights regarding market volatility and currency trends.

## Technologies Used
* **Data Visualization:** Microsoft Power BI
* **Data Source:** Historical currency exchange datasets (2014-2024)
* **Methodology:** Time-series analysis and statistical computation
